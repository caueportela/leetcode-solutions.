# 179. Largest Number



# Given a list of non-negative integers `nums`, arrange them such that they form the largest number and return it.

Since the result may be very large, return it as a string.

Example 1:

Input: nums = [10,2]  
Output: "210"

Example 2:

Input: nums = [3,30,34,5,9]  
Output: "9534330"



Constraints:

1 <= nums.length <= 100  
0 <= nums[i] <= 10⁹  



Test cases:

Test 1:  

Input: nums = [1]  
Output: "1"  

Test 2:  

Input: nums = [10]  
Output: "10"  

Test 3:  

Input: nums = [0,0]  
Output: "0"  

Test 4:  

Input: nums = [121,12]  
Output: "12121"  

Test 5:  

Input: nums = [824,938,1399,5607,6973,5703,9609,4398,8247]  
Output: "9609938824824769735703560743981399"
